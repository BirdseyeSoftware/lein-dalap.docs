lein-dalap

Clojure to Clojurescript transformer.

Get Started! » Contribute »

Same Codebase

Don't repeat yourself doing small changes to your source code in order to be used in Clojurescript. Use the same codebase for both versions (Clojure and Clojurescript) of your project.

Uses Clojure Reader

By using the powerful Clojure reader, we transform your source code easily; you just need to add a few meta tags, specify some rules CSS style and lein-dalap will do the rest.

Sane Defaults

lein-dalap already offers some common rule transformations that you are likely going to use in your code, replacing Java common types with javascript common types, gneral clojure to cljs namespace transformations, etc.
