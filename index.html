lein-dalap

Clojure to Clojurescript transformer.

Get Started! » Contribute »

Same Codebase

No need to fork your Clojure project in order to make it Clojurescript compatible. Use the same codebase for both versions of your project.

Uses Clojure Reader

By using the powerful Clojure reader, we transform your source code reliably; you just need to add a few meta tags, specify some CSS-like rules and lein-dalap will do the rest.

Sane Defaults

lein-dalap offers out of the box rule transformations that you are likely going to use in your code, replacing Java common types with javascript ones, general clojure to cljs namespace transformations, etc.
